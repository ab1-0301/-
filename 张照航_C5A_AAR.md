# C5A 挑战复盘（AAR）

## 1. 我做了什么？
简要描述你在 C5A 挑战中完成的任务（Level 1-3 或 Level 4）：
- 注册 GitHub 账号，创建个人仓库
- Fork `neolaf2/neoskills` 仓库
- 克隆到本地（或通过网页操作）
- 分析仓库结构，撰写仓库分析文档
- 创建 `my-contribution` 分支，修改 README.md 并提交
- （可选）发起 Pull Request

## 2. 我学到了什么？
- 学会了 GitHub 的基本操作（fork, clone, branch, commit, push, PR）
- 理解了开源仓库的目录结构设计
- 掌握了 Markdown 表格和文档写作
- 知道了如何进行技术分析（技能的作用、输入输出）

## 3. 我是如何验证学习成果的？
- 成功在个人仓库中提交了分析文档
- 在 `neoskills` 仓库中创建了分支并留下了 commit 记录
- （可选）成功提交了 PR，仓库显示 `1 commit ahead`
- 截图保存了各个关键步骤

## 4. 遇到的困难及解决方法
| 困难 | 解决方法 |
|------|----------|
| 命令行报错 `command not found` | 改用网页操作 |
| Commit 按钮灰色 | 发现需要填写标题 |
| 不知道如何创建文件夹 | 学习在网页上通过文件名加路径创建 |
| 找不到 commit 历史 | 点击分支下拉菜单或直接访问 `commits/分支名` |

## 5. 下次如何做得更好？
- 提前熟悉 GitHub 网页操作流程，减少试错
- 写文档时先列大纲，再补充细节
- 多利用截图和日志记录过程，方便复盘
