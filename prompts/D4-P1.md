# D4-P1 总结通信工程中的 OFDM 技术

## Prompt
你是一名通信工程教授。请用 3 个要点总结 OFDM 技术的基本原理和优势，面向大二学生，控制在 150 字以内。

## Response
并行传输：将高速数据流拆分为多个低速子流，在相互正交的子载波上同时发送，延长符号周期，降低多径干扰。

频谱高效：子载波频谱允许重叠且互不干扰，相比传统频分复用大幅提升频谱利用率。

抗衰落简单：通过插入循环前缀（CP）消除符号间干扰，结合FFT快速实现调制解调，对抗频率选择性衰落。
