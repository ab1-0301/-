# 通信原理 Anki 抽认卡
# 导入方式：Anki → 文件 → 导入 → 选择此文件 → 字段分隔符: Tab → 笔记本: 选择或新建
# 格式：正面\t背面\t标签

通信系统的三大组成部分	信源、信道、信宿	通信原理
模拟信号与数字信号的区别	模拟信号：连续取值（如语音）；数字信号：离散取值（如二进制 0/1）	通信原理
奈奎斯特定理	无噪声信道下，最大码元传输速率为 2B（B 为带宽），单位 Baud	通信原理
香农公式	C = B·log₂(1 + S/N)，其中 C 为信道容量（bit/s），B 为带宽（Hz），S/N 为信噪比	通信原理
抽样定理（奈奎斯特准则）	fs ≥ 2fmax，即抽样频率至少为信号最高频率的两倍才能无失真重建	通信原理
量化	将连续抽样值映射为离散电平的过程，分为均匀量化和非均匀量化	通信原理
PCM（脉冲编码调制）	抽样 → 量化 → 编码，将模拟信号转换为数字信号的经典方法	通信原理
ASK（幅移键控）	用载波幅度代表数字 0/1，如 0→A=0，1→A=A₀	通信原理
FSK（频移键控）	用载波频率代表数字 0/1，如 0→f₀，1→f₁	通信原理
PSK（相移键控）	用载波相位代表数字 0/1，如 0→0°，1→180°	通信原理
QPSK（四相移键控）	用 4 种相位表示 2bit 组合：00→0°，01→90°，11→180°，10→270°	通信原理
QAM（正交幅度调制）	同时调制载波的幅度和相位，如 16QAM 用 16 种状态表示 4bit	通信原理
OFDM（正交频分复用）	将信道划分为多个正交子载波并行传输，抗多径衰落，用于 4G/5G/WiFi	通信原理
FDM（频分复用）	不同信号占用不同频率段同时传输	通信原理
TDM（时分复用）	不同信号分时占用同一信道	通信原理
CDMA（码分多址）	不同用户用不同正交码扩频，共享同一频率和时间	通信原理
汉明码	一种线性分组码，能纠正 1 位错误，检测 2 位错误	通信原理
卷积码	编码输出不仅与当前输入有关，还与之前状态有关，常用 Viterbi 译码	通信原理
LDPC 码（低密度奇偶校验码）	接近香农极限的信道编码，用于 5G、DVB、WiFi	通信原理
码间串扰（ISI）	相邻码元间因信道不理想产生的干扰，可用均衡器消除	通信原理
升余弦滚降滤波器	用于减少码间串扰的脉冲成型滤波器，滚降系数 α 决定带宽	通信原理
扩频通信	用远大于信号带宽的伪随机码扩展信号频谱，抗干扰强	通信原理
分集接收	通过多个独立路径接收同一信号来对抗衰落，如空间分集、频率分集	通信原理
信噪比（SNR）	信号功率与噪声功率之比，单位 dB，公式：SNR_dB = 10·log₁₀(S/N)	通信原理
误比特率（BER）	错误比特数占总传输比特数的比例，衡量数字通信系统性能的核心指标	通信原理
眼图	用示波器观察数字信号波形，张开越大说明码间串扰越小	通信原理
载波同步	接收端需要恢复与发送端同频同相的载波进行相干解调	通信原理
帧同步	在数字通信中识别数据帧的起始位置	通信原理
多径效应	信号经多条路径到达接收端，产生时间扩散和频率选择性衰落	通信原理
均衡器	补偿信道畸变的滤波器，消除码间串扰	通信原理
